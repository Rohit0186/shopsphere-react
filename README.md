* 🔍 **Smart Product Fetch**: Real-time integration with Fakestore API.

* 🔎 **Optimized Search**: Case-insensitive filtering logic separated from the UI components.

## 🧠 Key Technical Highlights

* **UI ≠ Logic:** Search and filter logic is kept inside the page components, keeping the Navbar purely for UI interaction.
* **Predictable State:** Switched from basic `useState` to `useReducer` for complex cart transitions to ensure state predictability.
* **Performance:** Optimized CSS-in-JS using Tailwind v4's high-speed PostCSS bundling.
* **Maintainability:** Pure utility functions for price math make the app easy to scale or change currency factors.
